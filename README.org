* Dotfiles

  You are your dotfiles.

** Predependencies
    - zplug
    #+BEGIN_SRC sh
    git clone https://github.com/zplug/zplug ~/.zplug
    #+END_SRC
    - spacemacs
    #+BEGIN_SRC sh
    git clone https://github.com/syl20bnr/spacemacs ~/.emacs.d
    #+END_SRC

** Install
   #+BEGIN_SRC sh
   git clone https://github.com/eggcaker/dotfiles ~/.dotfiles
   cd ~/dotfiles
   make install
   #+END_SRC
